In this project, I developed a neural network model for musical instrument classification using deep learning techniques. The goal was to analyze audio samples and accurately identify the instrument being played, such as trumpet, violin, or piano.

Key Highlights:

  • Audio Feature Extraction: Used Librosa to extract Mel Frequency Cepstral Coefficients (MFCCs), a crucial feature for sound analysis.
  • Deep Learning Model: Built a neural network with TensorFlow and Keras, including a fully connected layer with 256 hidden units and a softmax classifier for multi-class classification.
  • Performance Optimization: Experimented with different learning rates, batch sizes, and optimization strategies to improve accuracy.
  • Energy Efficiency Analysis: Integrated Zeus-ML to measure GPU energy consumption, balancing model performance with computational cost.
  • Achieved Accuracy: Over 99% validation accuracy, demonstrating the model's effectiveness in recognizing musical instruments.

This project provided hands-on experience with machine learning for audio processing, reinforcing my skills in deep learning, data preprocessing, and model optimization.
